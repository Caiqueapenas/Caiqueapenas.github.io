QUIZ Script
O que é?

O QUIZ Script é um jogo de perguntas para quem está aprendendo a linguagem de programação JavaScript e/ou para aqueles que já a dominam e querem testar seus conhecimentos.


O objetivo!

Expandir os seus conhecimentos sobre a linguagem, por meio de acertos e (principalmente) erros das questões, de modo que o QUIZ mostre quais assuntos o jogador deve revisar, desde o básico até o avançado.
